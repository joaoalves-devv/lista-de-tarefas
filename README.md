# 📝 Lista de Tarefas

Um projeto simples e funcional de **lista de tarefas** desenvolvido com **HTML, CSS e JavaScript puro**, com persistência de dados via **LocalStorage**.  
Permite adicionar, editar, remover e ordenar tarefas, além de possuir alertas personalizados com **SweetAlert** para melhor experiência do usuário.

---

## 🚀 Funcionalidades

- ➕ **Adicionar tarefas** facilmente (inclusive pressionando **Enter**)
- 🗑️ **Remover tarefas** individualmente
- ✏️ **Editar tarefas**
- 🔄 **Ordenar tarefas** (ascendente/descendente)
- 💾 **Salvar automaticamente** no `localStorage`
- ⚠️ **Alertas elegantes** com SweetAlert
- 🧹 **Botão “Limpar lista”** aparece apenas quando há tarefas

---

## 🧠 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **SweetAlert2** (para os alertas)
- **LocalStorage** (para salvar as tarefas localmente)
